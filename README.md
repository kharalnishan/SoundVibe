## Pages

2. **About** (`about.html`) - Company mission, story, values, and team

## Technology Stack

- **HTML5** - Semantic markup with ARIA accessibility
- **CSS3** - Modern layouts with responsive design
  - CSS Grid & Flexbox
  - CSS Variables for theming
  - Transitions, Transforms, Gradients, Shadows
  - Media queries for responsiveness
- **JavaScript (Vanilla)** - No frameworks or dependencies
  - Form validation with real-time feedback
  - Interactive image gallery with keyboard navigation
  - Responsive hamburger menu
  - LocalStorage for favorites management

## Deployment

### Deploy to Netlify

1. Push to GitHub:
   ```bash
   git add .
   git commit -m "Deploy SoundVibe"
   git push
   ```

2. Connect to Netlify:
   - Go to https://app.netlify.com
   - Click "New site from Git"
   - Select your GitHub repository
   - Netlify will auto-detect settings and deploy

3. Live URL will be provided by Netlify
